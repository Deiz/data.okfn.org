Tools and Plugins for working with Data Packages

General Tools

Data Package Viewer

View any Data Package online using our online Data Package viewer tool.

Data Package Creator

Creating a Data Package means creating a datapackage.json file. We provide an online application which will auto create your datapackage.json from a provided data file.

Git Data Package Viewer

The ODI's Git Data Viewer can preview Data Packages stored in git repositories. See also the blog post and source code.

Using with R and R Client

R Open Data Protocols Library with support for Data Packages and Simple Data Format

Using with Postgresql

There's a python script (with no dependenciess) to load a Simple Data Format Data Package into Postgresql

Using with SQLite

There is a simple python script (no dependencies) to load a Simple Data Format Data Package into SQLite

Libraries

datapackage (python module)

A module useful for loading and managing data packages in Python. The module is available in PyPI so it can be installed using:

pip install datapackage

The source code is a available under the GPL.

R Library

R Library with support for Data Packages and Simple Data Format
